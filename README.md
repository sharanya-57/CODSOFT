# CODSOFT
# 🚢 Titanic Survival Prediction (Task-01)

## 📌 Objective
To predict whether a passenger survived the Titanic disaster using machine learning techniques.

## 📊 Dataset
The dataset is taken from Kaggle and includes features like:
- Age
- Gender
- Passenger Class
- Fare
- Embarked

## ⚙️ Workflow
- Data Cleaning  
- Exploratory Data Analysis (EDA)  
- Feature Encoding  
- Model Training  
- Model Evaluation  

## 🤖 Model Used
- Logistic Regression  

## 📈 Results
- Achieved good accuracy (~80–85%)  
- Evaluated using Confusion Matrix  

## 🛠️ Tools & Technologies
- Python  
- Pandas  
- Seaborn  
- Matplotlib  
- Scikit-learn  
- Jupyter Notebook  

## 🎯 Conclusion
The model successfully predicts passenger survival.  
Key factors influencing survival include gender and passenger class.
